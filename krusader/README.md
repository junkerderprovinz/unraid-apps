# Krusader — Unraid template

Twin-pane KDE file manager (Krusader) with a native **dark theme**, served as a smooth,
hardware-accelerated **KasmVNC** web desktop. This folder holds only the **Unraid Community
Applications template** (`krusader.xml` + icon); the image, Dockerfile and full documentation
live in the source repo.

- **Source & full README:** https://github.com/junkerderprovinz/krusader
- **Image:** `junkerderprovinz/krusader` (GHCR + Docker Hub) · amd64 + arm64
- **Support thread:** https://forums.unraid.net/topic/198816-support-junkerderprovinz-krusader/

## Install

On Unraid open **Apps** (Community Applications) and search **Krusader**.
Manual: paste the raw template URL into **Add Container → Template**:

```
https://raw.githubusercontent.com/junkerderprovinz/unraid-docker-templates/main/krusader/krusader.xml
```
